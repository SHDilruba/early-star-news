What's the difference between let, var and const?

Let statement is block-scoped and not a Global scope local variable used for initializing a statement. This statement can be used in function as it is function-scoped and one of the main character of let statement is that it can be reassigned. Var statement is also used for variable declaration and it is Global scoped variable. Var is defined throughout the function but not if its declared in a block as it is not block-scoped.Var statement can be reassigned. Const is quite similar to Let with the only difference that once the variable has been defined then its value cannot be changed. #if the user tries to change the value, they will get an error.

What's the difference between arrow function and regular function?

The first major difference between arrow functions and regular functions is that arrow functions are always anonymous. They cannot be named. One of the best things about an arrow function is implicit returns. Implicit returns from arrow functions becomes especially useful when working with array methods or callbacks. Whereas a regular function created with the function keyword can be named or anonymous (function myFunc() {... } vs function() { ... }), arrow functions are always anonymous.This is a longer card with supporting text below as a natural lead-in to additional content.
